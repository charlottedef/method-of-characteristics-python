# method-of-characteristics-python
Python reimplementations of selected Method of Characteristics examples from the chapter "Method of Characteristics for the Efficient Simulation of Population Balance Models" (1D and 2D cases).
## Source

This repository contains independent Python reimplementations of selected examples from:

Zhu, X., Zhou, L., Braatz, R.D. (2022).
Method of Characteristics for the Efficient Simulation of Population Balance Models.
In: Fytopoulos, A., Ramachandran, R., Pardalos, P.M. (eds),
Optimization of Pharmaceutical Processes.
Springer Optimization and Its Applications, vol. 189.
https://doi.org/10.1007/978-3-030-90924-6
